# Uniform Images Directory

This directory contains images for uniform catalog items.

## Image Naming Convention

Images should be named using the following format:
```
{category}-{gender}.jpg
```

## Required Images

Based on the catalog categories and genders, you need the following images:

### Shirts
- `shirt-male.jpg` - Professional men's airline formal shirt
- `shirt-female.jpg` - Professional women's airline blouse
- `shirt-unisex.jpg` - Unisex airline shirt

### Pants/Trousers
- `pant-male.jpg` - Professional men's airline trousers
- `pant-female.jpg` - Professional women's airline trousers
- `pant-unisex.jpg` - Unisex airline trousers
- `trouser-male.jpg` - Alternative name for pants (male)
- `trouser-female.jpg` - Alternative name for pants (female)
- `trouser-unisex.jpg` - Alternative name for pants (unisex)

### Jackets
- `jacket-male.jpg` - Professional men's airline navy blazer
- `jacket-female.jpg` - Professional women's airline blazer
- `jacket-unisex.jpg` - Unisex airline jacket

### Shoes
- `shoe-male.jpg` - Professional men's airline black formal shoes
- `shoe-female.jpg` - Professional women's airline formal shoes
- `shoe-unisex.jpg` - Unisex airline shoes

### Optional
- `default.jpg` - Default image to use as fallback if specific image is not found

## Image Specifications

- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 1200x1200 pixels (square format)
- **Quality**: High resolution for professional appearance
- **Style**: Professional airline uniform images matching Indigo Airlines aesthetic (navy blue, white, formal)
